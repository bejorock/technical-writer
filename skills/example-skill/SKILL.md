# Technical Writing Skill

Use this skill when the user asks about technical writing best practices.

## When to Use

- User asks about writing documentation
- User needs help with style guides
- User wants to improve clarity of technical content

## Guidelines

1. Use clear, concise language
2. Avoid jargon when possible
3. Use active voice
4. Structure content with headings
5. Include code examples when relevant

## Steps

1. Analyze the user's writing request
2. Apply technical writing best practices
3. Provide suggestions for improvement
